Sauvegarde des données de traduction  

But:

L'objectif de la fonctionnalité de sauvegarde des données de traduction est de copier ou de sauvegarder toutes les données de traduction de tous vos projets de récit vers un autre emplacement. Les données d'une histoire incluent tous les fichiers audio, image et texte sur lesquels vous avez travaillé. La création d'une sauvegarde de cette manière vous assurera d'avoir toujours une copie de tout le travail de traduction que vous avez effectué si quelque chose devait arriver à votre appareil.

L'autre emplacement où vous sauvegardez vos données de traduction peut être soit un lecteur flash OnTheGo (OTG), une (autre) carte SD, un ordinateur portable ou un lecteur en ligne.

De plus, cette fonctionnalité vous permettra de restaurer une sauvegarde enregistrée dans votre dossier SPWorkspace ou de partager des projets d'histoire avec un autre dossier SPWorkspace sur un autre appareil.

REMARQUE : Cela ne copie pas les vidéos que vous avez créées. Si vous souhaitez copier/sauvegarder/partager vos vidéos, revenez au menu principal et utilisez la fonction Copier/Partager des fichiers vidéo .

Comment conserver les données de traduction de votre histoire :  

Pour sauvegarder  votre ou vos projets d'histoire :

  1. sélectionnez le ou les récits dans la liste que vous souhaitez sauvegarder
  2. appuyez sur le bouton SAUVEGARDE.
  3. naviguez pour choisir un dossier dans lequel vous souhaitez que les fichiers du projet de récit soient copiés ou sauvegardés. Vous pouvez sélectionner un dossier à partir de la mémoire interne du téléphone, de la carte SD du téléphone ou d'un appareil auquel vous êtes connecté, tel qu'un lecteur flash OnTheGo. Il est recommandé de sauvegarder vos fichiers sur un lecteur, une carte mémoire ou un périphérique différent de celui qui contient votre dossier SPworkspace.
  4. Appuyez sur le bouton UTILISER CE DOSSIER.

C'est une bonne idée de sauvegarder vos données de traduction régulièrement (c'est-à-dire tous les trois mois) pendant que vous traduisez des histoires au cas où votre appareil ou votre carte SD tomberait en panne.

Pour restaurer  un projet qui a été sauvegardé :

  1. appuyez sur le bouton RESTORE (sans histoire sélectionnée)
  2. parcourez pour sélectionner le dossier qui contient l'histoire que vous souhaitez restaurer
  3. appuyez sur le bouton UTILISER CE DOSSIER

Si vous sélectionnez des projets d'histoire, puis appuyez sur le bouton SUPPRIMER , ces projets d'histoire seront complètement supprimés. (Si des vidéos ont été créées pour ces récits, elles ne seront pas supprimées. Toutes les vidéos créées seront enregistrées et disponibles dans le dossier SPworkspace/Videos.) Vous pouvez supprimer des projets de récit (1) s'ils n'ont pas été entièrement téléchargés ; (2) s'ils ne s'affichent pas correctement dans l'écran Modèles d'histoire ; (3) si l'espace est limité sur votre appareil ou sur la carte SD.

Pour partager des projets d'histoire avec un autre/deuxième appareil SPAdv :

  1. faire une sauvegarde du périphérique principal vers un lecteur externe
  2. branchez ce lecteur externe dans un deuxième appareil sur lequel le SPAdv est installé
  3. ouvrez l'application SPAdv sur ce deuxième appareil
  4. ouvrir la fonctionnalité de sauvegarde des données de traduction à partir du menu principal
  5. Appuyez sur le bouton RESTAURER
  6. accédez au lecteur externe
  7. sélectionnez les projets de story que vous souhaitez partager
  8. appuyez sur le bouton UTILISER CE DOSSIER.
  9. le ou les projets de récit restaurés seront ajoutés à votre dossier SPworkspace et apparaîtront désormais dans la liste des récits
