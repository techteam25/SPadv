Finaliser

Qui : Le traducteur principal ou le gestionnaire de téléphone ainsi que toute autre personne intéressée par la production vidéo serons l’acteur principale a cette étape de finalisaion.

 

But : Le but de la phase Finaliser est d'insérer des noms pour compléter le générique local glisser, prendre les décisions finales sur les nouvelles vidéos qui seront créées, et créer des vidéos pour les différents publics avec lesquels vous partagerez les vidéos.

 

Activités :

1.     Complétez la diapositive Crédits locaux : Appuyez sur le bouton MODIFIER LES CRÉDITS LOCAUX pour ouvrir une fenêtre contextuelle modifiable. Tapez les noms des personnes qui ont participé aux diverses activités de traduction, de composition de chansons, de photographie, de doublage, etc. Ajoutez un numéro de téléphone ou un adresse e-mail que les personnes peuvent contacter si elles souhaitent obtenir plus de vidéos ou si elles ont des questions sur l’histoire. Appuyez sur ENREGISTRER pour fermer la fenêtre et revenir à Finaliser. La coche/coche grise dans le cercle deviendra verte lorsque la diapositive de crédits locaux aura été modifiée.

2.     Tapez un nom d'histoire courte (dans votre langue locale) sur la ligne appropriée. Cela apparaîtra dans le nom de fichier de la vidéo pour vous aider à identifier le contenu de la vidéo. (Le numéro de l'histoire sera automatiquement ajouté au nom du fichier vidéo.) Ce texte peut être modifié à tout moment. La coche/coche grise dans le cercle deviendra verte lorsque vous aurez saisi du texte sur cette ligne.

3.     Choisissez les options à inclure dans la vidéo :

 Appuyez sur la case carrée de chaque option pour l'activer. Appuyez à nouveau dessus pour ne pas inclure cette option dans la vidéo que vous allez créer.

4.     Créez votre nouvelle vidéo :

Appuyez sur le bouton CRÉER UNE VIDÉO et attendez que l'application du téléphone génère les nouvelles vidéos. Ne faites pas d'autres activités avec le téléphone pendant la création de la vidéo. Cela prendra 3 à 30 minutes selon la longueur de l'histoire et de la chanson. Vous pouvez regarder la barre de progression pour voir la progression du  chargement de fichier. L'application créera automatiquement et simultanément votre histoire dans deux types de formats vidéo : mp4 et 3gp.

o    Le format 3gp est une vidéo basse résolution créée pour être lue sur les petits écrans des téléphones polyvalents. Ces vidéos sont relativement petites et n'occupent pas trop d'espace de stockage - en moyenne environ 20 Mo par vidéo d'histoire.)

o    Le format mp4 est une vidéo de résolution supérieure créée pour être visionnée sur des écrans plus grands et également projetée. Ces vidéos sont plus volumineuses et nécessitent chacune en moyenne 200 Mo de stockage.

5.     Recréez votre vidéo . Vous avez la possibilité de créer plusieurs versions de la vidéo de votre histoire. Sélectionnez différentes options ou fonctionnalités pour produire plusieurs versions de votre histoire.

 

Remarques :

  • Le nom du fichier de chaque vidéo sera généré automatiquement par l'application : un numéro à 3 chiffres issu du numéro du modèle d'histoire + un nom d'histoire courte dans la langue saisie par l'utilisateur lors de la phase Finaliser + un ethnologue à 3 lettres code de la langue tiré des informations sur la langue d'enregistrement + divers codes courts à 2 lettres indiquant les options ou fonctionnalités présentes dans la vidéo ( Fx = musique de fond et effets ; Sg = chanson locale ; Px = images ; Mv = mouvement sur les images ; Tx = texte de l'histoire / sous-titres)
  • Une vidéo basse résolution (3gp) prendra moins de place (elle sera plus petite), mais elle sera très « granuleuse » ou « pixélisée » sur un écran de smartphone.
  • Une vidéo de plus haute résolution (mp4) est nécessaire pour pouvoir lire les légendes/transcriptions de texte à l'écran.
  • Il existe plusieurs options de création audiovisuelle :
    • Film - diapositives animées avec narration audio (avec ou sans musique de fond)
    • Livre parlant - diapositives fixes avec narration audio et légendes/transcriptions de texte (avec ou sans musique de fond)
    • Livre - diapositives fixes avec légendes de texte (avec ou sans musique de fond) et changements de page automatiques
    • Radio - narration audio avec musique de fond mais pas de visuels
    • Tout ce qui précède avec ou sans chanson locale en annexe
