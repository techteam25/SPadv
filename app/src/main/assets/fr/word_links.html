Liste de liens de mots

Qui : Traducteurs et vérificateurs d'exactitude trouveront cet outil ou cette fonctionnalité utile à utiliser.

Finalité : plusieurs finalité du liste de lien de mots sont tel que :

  • pour fournir un accès pratique aux mots et aux noms difficiles dans toutes les histoires
  • pour créer une base de données de rendus audio en langue locale pour les mots et les noms difficiles en un seul endroit
  • pour aider les traducteurs à rendre les termes clés bibliques de manière cohérente et à prononcer les noms de personnes et de lieux de manière cohérente dans / à travers toutes les histoires
  • fournir des rétrotraductions textuelles littérales pour tous les mots et noms difficiles (et les translittérations de noms) comme un autre moyen d'aider un vérificateur d'exactitude dans son travail.

Activités :

1.     Dans le menu de la barre latérale,  la liste de lien de mots :

a.      Tapez dans la barre de recherche pour trouver un terme ou un nom spécifique, ou faites défiler les mots alphabétiques pour trouver ce que vous recherchez.

b.     Appuyez sur un mot/nom pour ouvrir la fenêtre contextuelle de ce terme.

2.     Appuyez sur un terme hyperlien de la phase TRADUIRE ou VÉRIFICATION DE LA PRÉCISION, section de texte de chaque diapositive.

a.      Un hyperlien bleu indique qu'il n'y a pas de rendu audio pour ce terme. Appuyez sur un hyperlien bleu pour diriger l'utilisateur vers l'écran de recherche du terme.

b.     Un hyperlien clair (texte normal souligné) indique qu'un rendu audio (au moins un) a été fait pour ce terme. En appuyant sur un lien hypertexte clair, l'utilisateur accèdera à l'écran de documentation audio et textuelle du terme. Faites glisser vers le haut pour voir l'écran de recherche.

3.     Lire et rechercher le sens d'un terme. Discutez de la façon de rendre le terme dans le contexte de l'histoire dans la langue locale. Considérez d'autres termes apparentés mais différents.

4.     Documenter un terme ou un mot en langue locale .

a.      Appuyez sur le bouton du micro en bas de l'écran pour enregistrer une traduction audio du terme dans la langue locale. Appuyez sur le bouton carré pour arrêter l'enregistrement.

b.     Appuyez longuement sur le fichier audio pour saisir une rétrotraduction littérale mot à mot dans la langue source (LWC) ou si un nom a été enregistré, tapez une translittération (phonétique) du mot.

5.     Rappelez-vous comment un mot a été rendu. Appuyez sur un lien hypertexte clair pour ouvrir la fenêtre contextuelle de la  liste de lien de mots, l'écran de documentation. Appuyez sur le bouton de lecture triangulaire pour écouter comment un nom ou un terme a été rendu dans une autre histoire. Quittez la fenêtre contextuelle pour reprendre le travail de traduction.

6.     Quittez la fenêtre contextuelle liste de lien de mots en appuyant sur le X dans le coin supérieur droit de l'écran. Reprenez vos activités de traduction et de vérification. Ou parfois, appuyer sur la flèche de retour de votre téléphone vous ramènera d'où vous venez de venir.

Remarques :

Les notes de la base de données Espace des mots   sont destinées à être de nature non encyclopédique - juste quelques notes pour donner aux traducteurs une signification ou un contexte essentiel et de base. S'il existe une équipe de traduction des Écritures ou de traduction de la Bible dans la langue locale, celle-ci doit toujours être consultée afin qu'il y ait une cohérence dans la façon dont les termes et les noms sont rendus dans les histoires.

1.     Deux types de mots sont inclus dans la base de données la liste de lien de mots:

·         Les Termes clés bibliques. Certains de ces termes peuvent avoir plusieurs sens selon leur contexte. Les traducteurs peuvent avoir besoin d'enregistrer plusieurs rendus audio pour un terme clé et de discerner quel rendu utiliser dans quel contexte. Plusieurs rendus audio peuvent également devoir être enregistrés pour documenter différentes formes d'un terme (par exemple, la forme verbale et la forme nominale)

·         Les Noms de personnes et de lieux. L'objectif principal de l'enregistrement d'un rendu audio pour les noms est de fournir une prononciation cohérente dans toutes les histoires.

3.     Il existe plusieurs sections dans l’écran de recherche de la fenêtre contextuelle de la liste de lien de mots  pour aider le traducteur de l'histoire dans sa décision quant à la manière de rendre le terme :

a.      Le mot hyperlié de l'histoire suivi de ses différentes formes grammaticales. Ceux-ci seront en haut de la fenêtre contextuelle dans la section marron.

b.     Notes sur la signification -- brèves définitions ou informations générales.

c.      Termes apparentés (mais différents), hyperliens -- pour fournir des éléments de comparaison et de contraste, des recherches supplémentaires.

d.     Exemples de rendus d'autres langues, retraduits -- pour fournir des idées pratiques sur la façon dont les traducteurs d'histoires pourraient vouloir traduire le terme dans leur langue locale.

4.     La barre d'enregistrement au bas de l'écran se développera en un écran étendu sous l'écran de recherche une fois qu'un enregistrement aura été effectué pour un terme clé. Une fois que les rendus audio ont été enregistrés pour un terme, ils peuvent être examinés dans cet écran de documentation étendu -- le rendu audio dans la langue locale peut être écouté et le texte rétrotraduit associé peut être lu.

5.     La base de données de langue locale de  la liste de lien de mots pourrait être sauvegardée régulièrement.

Le dossier de la liste de lien de mots est situé à la racine de l’espace de travail SP répertoire (qui est peut-être stocké sur la carte SD).

 Pour la sauvegarde, copiez la liste de lien de mots  répertoire vers un autre lecteur ou une autre carte.

6.     Des mots peuvent-ils être ajoutés à la base de données la liste de lien de mots? Les mots peuvent être ajoutés à la base de données, mais pas depuis l'application SPAdv. Il s'agit d'un processus en plusieurs étapes pour supprimer la base de données, retransformer le fichier csv en fichier xlsx, ajouter les mots souhaités dans  les colonnes appropriées, enregistrer la base de données en tant que fichier csv, remettre le fichier csv dans le SP Répertoire Espace de travail/ Liste de lien de mots, puis ouvrez l'application SPAdv afin que l'espace de travail soit analysé à nouveau.
