Slide Tell-Back

 Siapa : Satu atau dua orang dwibahasa, yang mengerti bahasa Inggris (atau Perancis atau Spanyol atau bahasa umum dengan Accuracy Checker) akan memegang telepon dan melakukan pekerjaan untuk tahap ini. Sebaiknya para penerjemah balik ini tidak terlibat dalam penerjemahan cerita, Story Tell Back, atau terlibat dalam pemeriksaan komunitas.

 Tujuan : Tujuan dari fase Slide Tell Back adalah untuk mengirimkan penceritaan ulang audio atau teks (terjemahan kembali) dalam bahasa sumber untuk setiap slide ke Pemeriksa Konsultan Oral Jarak Jauh yang bekerja melalui aplikasi web ROCC. Menceritakan kembali arti literal dari setiap frasa slide demi frasa akan membantu Pemeriksa memahami seberapa akurat dan bermakna setiap slide diterjemahkan ke dalam bahasa lokal. File audio Slide Tell Back akan melengkapi Story Tell Back (terjemahan balik) dan memberikan Pemeriksa Akurasi versi yang lebih mendetail dari setiap draf terjemahan slide untuk dibandingkan dengan teks sumber.

 

Kegiatan:

1.     Penerjemah balik harus memutar setiap terjemahan slide dalam bahasa lokal, mungkin berhenti sejenak setelah setiap frasa untuk mengetuk tombol mik di bagian bawah layar untuk memberikan terjemahan arti sebenarnya dalam bahasa Gateway.

2.     Jeda rekaman slide tell back dengan mengetuk tombol jeda dua garis vertikal.

3.     Lanjutkan slide frase demi frase menceritakan kembali perekaman dengan mengetuk tombol mic+.

4.     Hentikan atau selesaikan perekaman dengan mengetuk tombol persegi.

5.     Jika Anda perlu memulai kembali dengan terjemahan slide back, ketuk tombol stop persegi untuk menyegarkan tombol mikrofon . Jika tombol mic tidak ada tanda + di sebelahnya, berarti sudah siap untuk memulai ulang rekaman. Jika tombol mic ada tanda + di sebelahnya, berarti sudah siap untuk melanjutkan rekaman audio yang sedang berlangsung.

6.     Untuk setiap slide, ketuk ikon panah atas di cloud untuk mengirim rekaman audio Slide Tell Back ke Pemeriksa Akurasi / ROCC. Saat ponsel memiliki konektivitas atau sedang online, file akan dikirim. Tanda centang akan menggantikan panah di cloud dan cloud akan berubah menjadi biru saat ROCC telah menerima file audio.

7.     Untuk menulis terjemahan balik teks untuk setiap slide, ketuk baris tepat di bawah gambar dan tombol putar untuk membuka keyboard.

8.     Untuk mengirim terjemahan balik teks , ketuk ikon pesawat kertas putih / panah penunjuk kanan di sebelah kanan area SMS. Ikon pesawat berubah warna saat teks telah dikirim. Terjemahan balik tekstual akan muncul sebagai teks berwarna merah muda terang dalam fase Pemeriksaan Jarak Jauh.

9.     Geser ke bawah ke fase Pemeriksaan Jarak Jauh atau gunakan menu tarik-turun di bagian atas layar untuk mengetuk fase Pemeriksaan Jarak Jauh.

10. Peringatan akan ditampilkan jika pengguna memasuki fase Pemeriksaan Jarak Jauh sebelum semua terjemahan slide mundur dikirim.

 

Catatan:

●     Mengetuk mikrofon saat terjemahan bahasa lokal diputar akan secara otomatis menjeda pemutaran terjemahan lokal. Selain itu, mengetuk tombol putar untuk melanjutkan mendengarkan terjemahan draf akan secara otomatis menghentikan perekaman slide tell back. Aplikasi tidak akan mengizinkan pemutaran dan perekaman audio dilakukan secara bersamaan.

●     Jika penerjemah balik mengetahui bahwa Pemeriksa Akurasi (ROCC) lebih memilih terjemahan balik teks, mereka dapat mengetik dan mengirimkan terjemahan balik tertulis alih-alih melakukan terjemahan balik rekaman audio. Terjemahan balik audio/lisan lebih mudah untuk tim bahasa, tetapi terjemahan balik teks tertulis lebih mudah untuk Pemeriksa Akurasi.

●     Terjemahan balik teks jauh lebih murah untuk dikirim ke ROCC daripada terjemahan balik audio/lisan. File teks jauh lebih kecil daripada file audio.

●     Saat ROCC menyetujui slide, tanda centang abu-abu akan berubah menjadi hijau untuk slide tersebut. ROCCer mengontrol semua persetujuan slide . Persetujuan slide dapat dibatalkan oleh ROCCer dan tanda centang akan kembali menjadi abu-abu. Saat semua slide disetujui, Voice Studio dan fase produksi akhir lainnya akan dibuka kuncinya.
