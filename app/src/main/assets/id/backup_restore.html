Cadangan Data Terjemahan  

Tujuan:

Tujuan dari fitur Pencadangan Data Terjemahan adalah untuk menyalin atau mencadangkan semua data terjemahan untuk semua proyek cerita Anda ke lokasi lain. Data untuk sebuah cerita mencakup semua file audio, gambar, dan teks yang telah Anda kerjakan. Membuat cadangan dengan cara ini akan memastikan Anda masih memiliki salinan semua pekerjaan terjemahan yang telah Anda selesaikan seandainya terjadi sesuatu pada perangkat Anda.

Lokasi lain tempat Anda mencadangkan data terjemahan Anda dapat berupa flash drive OnTheGo (OTG), kartu SD (nother), laptop, atau drive online.

Selain itu, fitur ini memungkinkan Anda memulihkan cadangan yang disimpan ke folder SPWorkspace Anda atau berbagi proyek cerita dengan folder SPWorkspace lain di perangkat lain.

CATATAN: Ini tidak menyalin video yang Anda buat. Jika Anda ingin menyalin/mencadangkan/membagikan video Anda, kembali ke menu utama dan gunakan fitur Salin/Bagikan File Video.

Cara menyimpan Data Terjemahan cerita Anda:

Untuk mencadangkan  proyek cerita Anda:

  1. pilih cerita dari daftar yang ingin Anda buat cadangannya
  2. ketuk tombol CADANGAN.
  3. telusuri untuk memilih folder tempat Anda ingin file proyek cerita disalin atau dicadangkan. Anda dapat memilih folder dari memori telepon internal, dari kartu SD di telepon, atau dari perangkat yang terhubung dengan Anda seperti flash drive OnTheGo. Anda disarankan untuk mencadangkan file Anda ke drive, kartu memori, atau perangkat yang berbeda dari yang berisi folder SPworkspace Anda.
  4. Ketuk tombol GUNAKAN FOLDER INI.

Sebaiknya cadangkan data terjemahan Anda secara teratur (yaitu setiap tiga bulan) saat Anda menerjemahkan cerita untuk berjaga-jaga jika perangkat atau kartu SD Anda gagal.

Untuk memulihkan  proyek yang telah dicadangkan:

  1. ketuk tombol KEMBALIKAN (tanpa cerita yang dipilih)
  2. telusuri untuk memilih folder yang berisi cerita yang ingin Anda pulihkan
  3. ketuk tombol GUNAKAN FOLDER INI.

Jika Anda memilih proyek cerita apa pun lalu mengetuk tombol HAPUS , proyek cerita tersebut akan dihapus sepenuhnya. (Jika video dibuat untuk story tersebut, video tersebut tidak akan dihapus. Semua video yang dibuat akan disimpan ke dan tersedia dari folder SPworkspace/Videos.) Anda mungkin ingin menghapus proyek story (1) jika tidak diunduh sepenuhnya; (2) jika tidak ditampilkan dengan benar di layar Template Cerita; (3) jika ruang terbatas pada perangkat atau kartu SD Anda.

Untuk berbagi  proyek cerita dengan perangkat SPadv lain/kedua:

  1. lakukan pencadangan dari perangkat utama ke drive eksternal
  2. colokkan drive eksternal itu ke perangkat kedua yang telah diinstal SPadv di dalamnya
  3. buka aplikasi SPadv di perangkat kedua itu
  4. buka fitur Translation Data Backup dari menu utama
  5. Ketuk tombol KEMBALIKAN
  6. telusuri ke drive eksternal
  7. pilih proyek cerita yang ingin Anda bagikan
  8. ketuk tombol GUNAKAN FOLDER INI.
  9. proyek cerita yang dipulihkan akan ditambahkan ke folder SPworkspace Anda dan sekarang akan muncul di daftar cerita
