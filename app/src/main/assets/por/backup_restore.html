Backup de dados de tradução

Propósito:

O objetivo do recurso Translation Data Backup é copiar ou fazer backup de todos os dados de tradução de todos os seus projetos de história para outro local. Os dados de uma história incluem todos os arquivos de áudio, imagem e texto nos quais você está trabalhando. Criar um backup dessa maneira garantirá que você ainda tenha uma cópia de todo o trabalho de tradução concluído, caso algo aconteça com seu dispositivo.

O outro local onde você faz backup de seus dados de tradução pode ser uma unidade flash OnTheGo (OTG), um (outro) cartão SD, um laptop ou uma unidade online.

Além disso, esse recurso permitirá que você restaure um backup salvo em sua pasta SPWorkspace ou compartilhe projetos de história com outra pasta SPWorkspace em outro dispositivo.

NOTA: Isso não copia os vídeos que você criou. Se você deseja copiar/fazer backup/compartilhar seus vídeos, retorne ao menu principal e use o recurso Copiar/Compartilhar arquivos de vídeo .

Como preservar os dados de tradução da sua história:

Para fazer backup do(s) seu(s) projeto(s) de história:

  1. selecione a(s) história(s) da lista que você deseja fazer backup
  2. toque no botão BACKUP.
  3. navegue para escolher uma pasta onde deseja que os arquivos do projeto de história sejam copiados ou copiados. Você pode selecionar uma pasta da memória interna do telefone, do cartão SD do telefone ou de um dispositivo ao qual esteja conectado, como uma unidade flash OnTheGo. É recomendável fazer backup de seus arquivos em uma unidade, cartão de memória ou dispositivo diferente daquele que contém sua pasta SPworkspace.
  4. Toque no botão USE ESTA PASTA.

É uma boa ideia fazer backup de seus dados de tradução regularmente (ou seja, a cada três meses) enquanto você traduz histórias, caso seu dispositivo ou cartão SD falhe.

Para restaurar  um projeto cujo backup foi feito:

  1. toque no botão RESTAURAR (sem história selecionada)
  2. navegue para selecionar a pasta que contém a história que você deseja restaurar
  3. toque no botão USE ESTA PASTA.

Se você selecionar qualquer projeto de história e tocar no botão EXCLUIR, esses projetos de história serão completamente excluídos. (Se os vídeos foram criados para essas histórias, eles não serão excluídos. Todos os vídeos criados serão salvos e disponibilizados na pasta SPworkspace/Vídeos.) Você pode querer excluir projetos de histórias (1) se eles não foram baixados completamente; (2) se eles não estiverem sendo exibidos corretamente na tela de modelos de história; (3) se o espaço for limitado em seu dispositivo ou cartão SD.

Para compartilhar  projetos de história com outro/segundo dispositivo SPadv:

  1. faça um backup do dispositivo principal para uma unidade externa
  2. conecte essa unidade externa a um segundo dispositivo que tenha o SPadv instalado
  3. abra o aplicativo SPadv nesse segundo dispositivo
  4. abra o recurso Translation Data Backup no menu principal
  5. Toque no botão RESTAURAR
  6. navegue até a unidade externa
  7. selecione os projetos de história que deseja compartilhar
  8. toque no botão USE ESTA PASTA.
  9. o(s) projeto(s) de história restaurado(s) serão adicionados à sua pasta SPworkspace e agora aparecerão na lista de história
