Translation Data Backup

Purpose : The purpose of the Translation Data Backup feature is to copy or backup your story project to another location. This could be either on your phone, an SD card, or an OnTheGo (OTG) flash drive, so that if something happens to your device, you will still have a copy of all the translation data work that you have done. The story project include all the audio and picture files that you have been working on.

In addition, this feature will allow you to restore a saved backup to your Workspace folder.

How to do it : To backup your story project, select the story project from the list and tap the BACKUP button. You will need to pick a folder where you want the story project files to be copied. You may select a folder from the internal phone memory, from the SD card in the phone, or from a device that you are connected to such as an OnTheGo flash drive. It is recommended that you backup your files to a different drive or memory card than the one that contains your Workspace folder.

It is a good idea to backup the story project regularly (i.e. once every two or three months) while a story is being edited just in case your device or SD card fails.

To restore a project that has been backed up, you tap the RESTORE button and then select that folder that contains the story to be restored and click the USE THIS FOLDER button.

Tapping the DELETE button allows the user to delete the stories that have been selected. Since space can be limited on a phone or SD card, this can be a useful way to free up some space. However, it is a good idea to backup the story to an external drive before deleting it to save any work that has been done so that it can be restored later.

You can share story projects with others by doing a backup to an external drive that can be passed to others who can use the restore feature on another phone.
